# L1 — High-Level Requirements

## L1-1: Data Mining Service

The platform shall provide a .NET background service that discovers meetups and events from internet sources (Eventbrite, Meetup.com, and general web search) where Black professionals working in the space or defense industry are likely to attend or speak.

## L1-2: Data Store

The platform shall persist discovered events and their metadata to a relational data store (SQL Server via Entity Framework Core) so that data survives service restarts and can be queried efficiently.

## L1-3: Web API

The platform shall expose a RESTful ASP.NET Core Web API using Controllers that allows consumers to search, filter, and retrieve stored events and related metadata.

## L1-4: Solution Structure

The platform shall be organized as a clean .NET solution with clearly separated projects for the API, services, data access, and shared models.
